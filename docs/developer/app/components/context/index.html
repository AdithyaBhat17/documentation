Objects#

steps

  1. app.context.steps is an object which contains key-value pairs of steps and the step values user inputs in the format step_name: step_value

  2. A new step is added into app.context.steps only after the validation of that step.

  3. When it hits an intent, this steps object is initially empty

"steps": {
"mobilenumber": "9876543210",
"verifyotp": "9222"
},
