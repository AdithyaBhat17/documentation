Setup and configuration

In this article you will learn how to configure email tickets. There are 3 steps to be followed in the given order:

  1. Create an email channel(email ID
  2. Link email channel to Inbox (with forwarding address
  3. Enable email ticketing from settings

After this configuration is successful, the tickets that are created via. Raise ticket node for email support groups will be assigned to a live agent for resolution.

1. Set up Email channel​

Configure the email channel from the channels module, follow the below steps:

  1. Open Channels > Email. You can create an email(/account) for your group that handles email tickets.

  2. Select Basic/Advance and add Email server, Username, Password, etc. Click here for detailed documentation.

  3. In Configuration object field, select Support ticketing.

  4. Save the changes.

  5. Once the email account details are Saved, Forwarding Address will be displayed.

info

Purpose of Forwarding address

When any new email is sent to this particular email ID (that was created for support ticketing), the user email will be forwarded to the current Inbox section. That is, the emails sent to the email ID will be available on the yellow.ai platform so that the Agent - customer communication can take place in the Inbox Email chat rather than Email chat.

2. Set mail forwarding​

To forward mails from the email chatbox to the yellow.ai platform, follow the below steps:

  1. Copy the Forwarding address from the channels page.
  2. Log in and open your mailbox (Gmail, Outlook, etc).

For Gmail

  1. Open All settings and go to Forwarding and POP/IMAP tab.

  2. Click Add a forwarding address and paste the copied address.

  3. An email verification code will be sent to the Inbox to ensure that the configuration is authorized. Open Inbox > Chats > Bot Messages.

  4. Copy the verification code from the Inbox and paste it into Gmail settings to enable the forwarding of messages.

3. Enable email ticketing settings​

The email ticketing toggle must be enabled from the Inbox settings. Explained in detail here
