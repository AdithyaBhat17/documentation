Send data to API

In this article, you will learn -

  1. How to send data from bot flow to an API
  2. How to pass URLs in an API parameter
  3. How to send JSON object in API parameter

1. Send data to API from a bot flow​

Once you configure an API as mentioned in add a new API, you can pass all the input parameter values in the bot flow where you want to consume it. These are usually dynamic values that you can capture directly from the user conversation.

  1. To send a dynamic data from a flow, rename the parameters with {{{parameterName}}} for the API in Studio > API and save it.

  1. Open the flow where you wish to send data using the user's input.
  2. Add the API actions node in the flow.

  1. Select the API you wish to add.

  1. Add the dynamic parameter fields (such as authentication, email, token, name etc. as per the API).

  2. Select a value (transformation function) in Parse API response to extract a part of API response before storing.

  3. Test the API to ensure it is working as desired.

  1. Use Store Response in to store the response in a variable to access it.

2. Use base64 URL conversion for files​

In cases like uploading an image, video, or any other file formats, you can convert images into a readable string that can be used or transferred anywhere. It is a common practice to upload files through URL through Base64 encoding.

Use the following code inside a body/params -

Pass URLFormat
Directly$$tobase64{url}
Through variable$$tobase64{{{{url}}}
