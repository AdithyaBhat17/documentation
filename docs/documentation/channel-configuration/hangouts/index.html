Google Hangouts

  1. Go to the link https://developers.google.com/hangouts/chat/how-tos/bots-publish and click Enable the API in step 1.

  1. Select Create a new project from the dropdown and enter the project name

  1. Open your project in the API CONSOLE by clicking the hyper link in step 3

  2. In Google API console click ENABLE API AND SERVICES

  1. Search for Hangouts Chat API and enable it.

  1. Make sure your project name is selected in the project section

  1. Click on your project name as shown below

  1. Go to Configuration settings and fill the required bot details.

Paste “https://app.yellowmessenger.com/integrations/hangouts/receive/botId” in Bot URL field.

Example : https://app.yellowmessenger.com/integrations/hangouts/receive/x157148753865834

  1. Go to https://chat.google.com/ , add your bot and start your conversation
